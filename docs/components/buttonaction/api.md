# ButtonAction

底部按钮弹出层组件。用于在底部有太多需要展示的事物时，提供一个弹出层。


## API

| 名称                |  类型           | 默认值           | 含义           |
| -------------      | ------------- | --------------- | --------------- |
| actions            | `array`  |         |  按钮标题，必填。 可设置按钮样式   |

当组件内有内容时，会自动产生 `more` 按钮，否则表现为固定在底部的按钮组
