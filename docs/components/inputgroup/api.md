# InputGroup

输入框组组件。

## 组件

### InputGroup

`<InputGroup>` 组件。

#### Props

##### `header`

> PropType: `string`

输入框组 title 

##### `newline`

> PropType: `bool`

输入框与文字两行布局

##### `fields`

> PropType: `array`

```js
fields: PropTypes.arrayOf(
          PropTypes.shape({
            label: PropTypes.string,
            placeholder: PropTypes.string
        })),
```

输入框组数据
